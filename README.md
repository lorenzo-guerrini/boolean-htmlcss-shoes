# boolean-htmlcss-shoes

## EN

Simple template for a shoes website, made with HTML and CSS, focusing on the behavior of viewport height, viewport width, and the `<a>` tag linked to specific tags on the page.

## IT

Semplice template per un sito web di scarpe, realizzato con HTML e CSS con l'obbiettivo di studiare il fuzionamento della viewport height, della viewport width, e del tag `<a>` con link a tag specifici nella pagina.
